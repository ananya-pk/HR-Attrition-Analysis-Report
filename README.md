# HR-Attrition-Analysis-Report
HR Attrition Analysis Dashboard built using Power BI to explore employee turnover trends across departments, job roles, salary, and demographics. Includes interactive visualizations and DAX-based KPIs to support data-driven HR decisions.

# Overview

This project presents an interactive HR Attrition Analysis Dashboard built using Power BI to analyze employee data and identify key factors influencing attrition.

The dashboard provides insights into employee turnover trends across departments, job roles, salary, tenure, and demographics, helping organizations make data-driven HR decisions.

# Key Objectives
Analyze employee attrition patterns
Identify factors affecting employee turnover
Compare salary and tenure across job roles
Provide actionable insights for improving retention

# Key Metrics (KPIs)
Total Employees
Attrition Rate
Average Monthly Income
Average Tenure
Bonus Distribution

# Dashboard Features
Attrition analysis by Department, Gender, and Job Role
Salary comparison across different roles
Employee distribution by Age Group and Seniority Level
Bonus analysis using Treemap visualization
Interactive filters (slicers) for dynamic data exploration

🛠️ Tools & Technologies Used
Power BI – Data Visualization
DAX – Calculated Measures
Power Query – Data Cleaning & Transformation

💡 Key Insights
Higher attrition observed in specific departments (e.g., Sales)
Mid-level employees show increased turnover trends
Salary and tenure play a significant role in retention
Younger age groups tend to have higher attrition rates

🚀 How to Use
Download the .pbix file from this repository
Open it using Power BI Desktop
Use filters/slicers to explore different insights

📌 Conclusion

This project demonstrates how data visualization and analytics can be used to understand workforce trends and support strategic HR decision-making.
